git学习

不掌握github,怎么好意思说自己是程序员..

分支branch

我理解的git分支就是帮我们复制当前所在工作目录的文件夹这一操作, 和我们普通的ctrl+c ctrl+v操作一样,无特殊之处,如果非要说特殊,就特殊在git可以记录我们文件的所有历史版本,所以它可以选定任意历史版本的文件夹进行复制.

git branch 查看分支

如果我们假象分支就是复制文件这个操作,那么要查看复制了多少个文件就用这个命令git branch. 因为Linux的思想是一切介文件,所以它这个复制动作其实就是新建一个文件,这个文件里记录了当前的目录及目录里面的文件. git branch就是列出所有复制这一类型的文件

git branch [branch_name] 新建分支

我们自己ctrl+c ctrl+v一样.让git帮我们把当前所在的分支文件复制一份.

通常情况下当前目录就是我们正在开发的目录,是最新的. 如果要为两个月前的某个版本进行复制该如何操作呢?看下个命令 checkout

git checkout [branch_name]切换分支

checkout类似于我们用cd命令切换文件夹差不多, 不同的是:它切换的是我们commit版本的文件夹.

回答上个问题:我们要分两步:

  1. checkout 到两个月前提交的那个版本下
  2. 执行 git branch 命令

git branch -d [branch_name]删除分支

新建分支一般都要合并.

分支小结

  • 记住三个指令: 新建, 删除, 切换
