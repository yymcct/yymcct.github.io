Server 对象

  • Server: 构造函数,没什么特别
  • Start:方法
    • 创建Socket对象,并监听
    • 定义一个线程池等待链接并处理:如果有请求则获取一个Connection对象,然后用Host对象去处理这个请求

  • OnRequestEnd:
    • 释放Connection对象
