Vue学习

安装

引入vue.js<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script> 即可

Vue 实例

  • 一个Vue应用由一个通过new Vue创建的根Vue实例,以及可选的,嵌套的,可复用的组件树组成.
  • 当Vue被创建时, 它将data对象中的所有属性加入到Vue的响应系统中. 当这些属性的值发生改变时,视图会产生响应, 即匹配更新为新的值
    • Vue 实例还暴漏了一些有用的实例属性与方法. 他们都有前缀$,以便与用户定义的属性区分开

模板语法

  • 插值使用Mustache语法:, 如果是Html特性上,应该使用v-bind指令,如果是布尔值,则表示该特性是否渲染,每个绑定只能包含单个javascript表达式
  • 关于指令参数: <a v-bind:href="url">...</a> 中v-bind是指令, href是参数, url是表达式, 告知v-bind将href和url的值进行绑定
  • v-前缀作为一种视觉提示,可以v-bind缩写是:, v-on缩写是@

知识点总结

  • Vue安装的方式有哪几种?
    答: scrtipt 直接引用, 或者脚手架安装

  • 一个Vue应用由一个通过(?)创建的根Vue实例,以及可选的,嵌套的,可复用的(?)组成.
    答: new Vue, 组件树

  • 当Vue被创建时, 它将(?)中的所有属性加入到Vue的响应系统中. 当这些属性的值发生改变时,视图会产生响应, 即匹配更新为新的值
    答: data对象

  • Vue 实例还暴漏了一些有用的实例属性与方法. 他们都有前缀(?),以便与用户定义的属性区分开
    答:$

  • Vue中给文本插值的语法是?
    答:

    • Vue中给特性插值的语法是?
      答 v-bind:href='url'

  • Vue 中v-bind的缩写是?
    答: :

  • Vue中v-on的缩写是?
    答: @

  • Vue中关于指令参数:<a v-bind:href="url">...</a> 中是指令,是参数, ___是表达式
    答: v-bind,href,url

  • Vue中使用v-bind,其表达式是布尔值, 则代表?
    答: 该特性是否进行渲染
